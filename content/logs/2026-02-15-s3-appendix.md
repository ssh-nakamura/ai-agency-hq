## 重要な判断・推奨

### チャンネルコンセプトの選定

**案A「AIだけで会社を作る実験レポート」を推奨**する理由：

1. **差別化**
   - 競合チャンネルは「使い方」を教える
   - 当プロジェクトは「実験そのもの」がコンテンツ

2. **信頼性**
   - 「本当の実験」なので、成功・失敗両方が興味深い
   - AI副業の甘い話より、リアルな過程のほうが信頼性高い

3. **展開性**
   - ShieldMe開発、コンテンツ事業、YouTube事業がすべてネタになる
   - ブログ記事、note、書籍への展開が容易

4. **学習価値**
   - 視聴者にとって「AIで事業を作る過程」を学べる
   - 「誰でも月10万稼げる」より社会的価値が高い

### ツール構成の選定

**推奨: ChatGPT API + ElevenLabs + InVideo AI + CapCut**

理由：
1. **実行可能**: 全て実績がある商用ツール
2. **コスト効率**: ¥8,000～12,000/月で高品質
3. **日本語対応**: InVideo（UI）、ElevenLabs（音声）とも日本語対応
4. **段階的導入**: 最初は無料VOICEVOX + ChatGPT APIで開始→後で品質向上

### 着手の優先順位

1. **今**: YouTube設計書の完成 ← ★完了★
2. **Week 1-2**: YouTubeチャンネル開設（株主承認後）
3. **Week 3**: 1本目動画の台本作成・制作開始
4. **Month 2**: 品質向上フェーズ（InVideo等の契約）
5. **Month 3～**: 週1ロング + 週2～3ショートの定期投稿開始

## 所感・学び

### Web調査の重要性

AIツール業界は変化が激しい。設計書執筆時点での「最良」が3ヶ月後に古くなる可能性がある。特に：

- **料金**: ElevenLabsが¥1,100/月から値上げされていた形跡あり（Webの記事に振幅あり）
- **機能**: InVideo AIが新しいテンプレート追加、Synthesia が日本語改善
- **競合**: HeyGen が日本語対応を強化しており、今後の選択肢

→ **対応**: 設計書に「3ヶ月ごとにツール比較を再実施」という注記を入れるべき

### 「実験レポート」の差別化力

競合分析で気づいたこと：

現在、YouTubeの「AI副業系」チャンネルは100以上存在する。ただし、**「AIだけで会社を作る実験」のドキュメンタリーは0～5本程度**。

このニッチは非常に狙い目。特に、「失敗も含めて全公開」という仮想機関AIの基本哲学は、他にはない強みになる。
