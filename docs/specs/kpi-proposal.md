### 1.1 競合調査レポート（市場研究-競合）のレビュー

**ファイル**: `docs/specs/market-research-competitors.md`

#### レビュー結果: ★★★★☆（良好、一部補強推奨）

| 項目 | 評価 | 指摘事項 |
|------|------|---------|
| 市場規模データ | ○ | MM総研データ（139億円）の出典は明確。最新追加調査により裏付け完了 |
| 競合分析 | ○ | 主要競合を網羅。価格帯の空白地帯の発見は戦略的価値が高い |
| 価格設定提案 | △ | 月額980円〜4,980円の根拠が不足。**要補強** |
| コスト構造検証 | △ | API単価の実測データがない。**Phase 1で検証必須** |
| ターゲット設定 | ○ | VTuber・配信者コミュニティは論理的 |

#### 指摘事項の詳細

**1. 価格設定の根拠不足**
- 月額980円という価格は「競合の1/5〜1/50」という相対的な根拠のみ
- **推奨**: 以下の要素を考慮した価格検証が必要
  - ユーザー1人あたりのAPI費用（実測値）
  - ターゲット層（個人インフルエンサー）の支払意思額
  - 競合サービスとの機能比較での相対的価値

**2. API費用の実測が未完了**
- コスト見積もりレポートでは「Haiku 1件あたり入力500トークン+出力100トークン」と仮定
- **リスク**: 実際のLLM判定で必要なトークン数が想定より多い場合、価格設定が成り立たない可能性
- **推奨アクション**: Phase 1開始前に100件の実測データを取得し、価格モデルを再検証

**3. 市場規模の最新データ補強**
- 本レビューで追加調査を実施し、2024年度の市場規模139億円（MM総研）を確認
- 出典: [MM総研プレスリリース](https://www.m2ri.jp/release/detail.html?id=553)、[日経クロステック報道](https://xtech.nikkei.com/atcl/nxt/mag/nc/18/020600010/091600124/)
- 補足: 検知系サービスは35.3億円、対策系サービスは42.1億円（2021年度）
- **結論**: 市場は成長中であり、ShieldMeの参入タイミングは妥当

#### 修正推奨事項

以下の追記を推奨する：
- 価格帯の根拠として「個人向けSaaS月額500〜3,000円が一般的」というベンチマークを追加
- API単価の実測データが揃うまで、価格は「仮設定」と明記
- 損益分岐点ユーザー数の計算式を明記（透明性向上）

## Part 2: KPI目標値の提案

### 2.1 KPI設定の基本方針

#### シナリオ設計思想

| シナリオ | 前提条件 | 用途 |
|---------|---------|------|
| 楽観 | 競合優位性が機能し、市場の反応が良好 | 上限目標、投資判断の最良ケース |
| 標準 | 計画通りに進行、平均的な市場反応 | **意思決定のベースライン** |
| 悲観 | 競合対抗や市場の反応が鈍い | リスク管理、撤退判断の基準 |

**重要**: 標準シナリオでの達成率80%以上を「順調」と評価する。悲観シナリオを3ヶ月連続で下回る場合は戦略見直しを提案する。

#### Phase定義（再掲）

- **Phase 0**: 基盤構築（現在）〜 サイト公開、ShieldMe仕様確定
- **Phase 1**: プロダクト開発+コンテンツ運用 〜 MVP完成、初収益発生
- **Phase 2**: 収益化 〜 黒字化達成

---

## Part 5: リスクと前提条件

### 5.1 主要リスク

| リスク | 発生確率 | 影響度 | 対策 |
|--------|---------|-------|------|
| API費用が想定の2倍 | 中 | 高 | Phase 1で実測、価格調整 or 撤退判断 |
| 競合の個人向けプラン参入 | 中 | 高 | 機能差別化、LLM精度で優位性確保 |
| YouTube収益化未達成 | 高 | 低 | 動画事業は補完的位置づけ、依存しない |
| コンテンツのネタ切れ | 低 | 中 | 事業運営自体がネタ、セッションログ活用 |
| 法的リスク（誹謗中傷誤判定） | 低 | 高 | legal部門の事前レビュー、免責条項 |

### 5.2 重要な前提条件

1. **Claude Max契約の継続**: 月¥30,000のコストは株主が負担継続
2. **X API Basicの導入判断**: Phase 1開始時に株主承認（月¥15,000）
3. **動画ツールの選定**: Phase 0で設計、Phase 1で導入（コスト株主承認）
4. **価格設定の柔軟性**: 実測データに基づき、Phase 1で価格調整可能とする
5. **ターゲット市場の検証**: VTuber・配信者の反応を見て、Phase 1で方向修正可能
