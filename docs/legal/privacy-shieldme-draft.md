## 第1条 本ポリシーについて

### 1.1 適用範囲
このプライバシーポリシーは、ShieldMeサービスのユーザーの個人情報取扱いについて定めています。本サービス利用時点で、本ポリシーに同意したものとみなします。

### 1.2 個人情報保護法への準拠
当社は、日本の「個人情報保護法」に基づき、ユーザーの個人情報を厳正に管理します。

## 第2条 取得する個人情報

### 2.1 Xアカウント認証時に取得する情報

ShieldMeにXアカウントを連携させる際、以下の情報を取得します：

| 情報カテゴリ | 具体例 | 取得方法 |
|------------|------|---------|
| **ユーザープロフィール** | ユーザーID、ユーザー名、表示名、プロフィール画像、自己紹介文 | X API（OAuth認証） |
| **言及・リプライ情報** | ユーザーへのリプライ投稿、引用リツイート | X API（タイムライン取得） |
| **フォロー関係** | フォロワー数、フォロー数、フォロワー・フォロー中ユーザー情報 | X API |
| **アカウント統計** | ツイート数、作成日時、言語設定 | X API |

### 2.2 サービス利用時に取得する情報

1. **ユーザーの操作ログ**
   - ログイン時刻、ログイン回数
   - 検知結果の表示・確認内容
   - フィードバック（「判定が正確」「判定が誤り」等）

2. **デバイス・接続情報**
   - IPアドレス
   - ブラウザの種類、OSの種類
   - アクセス日時

3. **決済情報**（有料プラン利用時のみ）
   - 支払い方法（カード番号は取得しません）
   - 請求先住所・メールアドレス
   - 支払い履歴

### 2.3 AI分析により推測される情報

ShieldMeのAIは、上記情報から以下の情報を推測・分類します：

- **心理状態の推測**（被害者のメンタル状態、ストレスレベル）
- **関心事・トピック**（職業、趣味、政治的見解等）
- **社会的属性**（推定年代、地域等）

**⚠️ 重要:** これらの推測情報は、プロファイリング（個人の性質を推測する行為）に該当する可能性があります。当社はこれを適切に取り扱い、マーケティング等の目的には利用しません。

## 第3条 個人情報の利用目的

### 3.1 主目的（ユーザーの明示的同意のもと）

1. **誹謗中傷の検知・分類**
   - AIを使用して、ユーザーへの誹謗中傷投稿を自動検知
   - 投稿を「強い非難」「軽微な批判」等のカテゴリに分類

2. **AIカウンセリング機能の提供**
   - ユーザーのメンタル状態を推測し、適切なメッセージを生成
   - メッセージはユーザーのデバイス内またはサーバーに保存

3. **ユーザーサポート・問い合わせ対応**
   - サポートメールへの返信、技術的な問題解決

### 3.2 副次目的（ユーザーの同意が必要な場合あり）

1. **サービス改善**
   - AI判定の精度向上のためのデータ分析
   - UX改善のためのユーザー行動分析

2. **統計情報の作成**
   - 年代別、地域別の誹謗中傷傾向の分析
   - **個人が特定できない形で集計**

3. **マーケティング・広告**
   - ユーザーの関心に基づいたサービス改善提案
   - **個人を特定した広告配信は行いません**

### 3.3 禁止される利用目的

当社は、以下の目的でユーザー情報を利用しません：

- ユーザーの政治的見解、思想の監視
- 信用調査、素行調査
- 警察等の公的機関への通報（ユーザーの同意なしに）
- 第三者への販売・転売

## 第4条 AI処理とデータの取扱い

### 4.1 AI処理の詳細

ShieldMeは以下のAIモデルを使用してデータを処理します：

| 処理項目 | 使用API/モデル | 処理場所 | 情報の種類 |
|---------|--------------|---------|----------|
| 誹謗中傷判定 | Claude API（Anthropic） | Anthropic のサーバー | テキスト投稿内容のみ |
| カウンセリング | Claude API（Anthropic） | Anthropic のサーバー | ユーザーのメンタル状態推測値 |

### 4.2 第三者提供（データ送信）

以下の第三者にユーザーデータが送信される可能性があります：

1. **Anthropic (Claude API)**
   - 送信データ：投稿テキスト、推測されるユーザーの心理状態
   - 目的：AI判定・分析
   - 個人情報保護：Anthropicのプライバシーポリシーに従う
   - Anthropic URL: https://www.anthropic.com/privacy

2. **X（旧Twitter）API経由**
   - 送信データ：ユーザープロフィール、言及投稿
   - 目的：データ取得の認証
   - XのAPI利用規約に従う

### 4.3 個人情報保護法上の位置付け

- **当社の位置付け**：個人情報取扱事業者（オンラインサービス提供者）
- **Anthropicの位置付け**：共同利用者または委託先（検討中）
- **X（Twitter）の位置付け**：APIプロバイダ、情報提供者

## 第5条 データの保存・保持

### 5.1 保存期間

| 情報カテゴリ | 保存期間 | 保存理由 |
|------------|--------|--------|
| Xプロフィール情報 | アクティブ利用期間中 | サービス提供 |
| 言及・リプライ投稿 | 最新100件 + 過去3ヶ月 | ユーザーの閲覧・確認用 |
| AI判定結果 | 1年間 | サービス品質改善、ユーザー参照用 |
| 利用ログ | 90日間 | セキュリティ監視、不正検知 |
| 決済情報 | 7年間 | 法令要件（税務・会計） |
| フィードバック情報 | 無期限 | サービス改善（個人情報を削除後） |

### 5.2 ユーザーアカウント削除時

1. ユーザーが本サービスのアカウントを削除した場合、以下の処理を行います：
   - プロフィール情報、投稿データ：即座に削除
   - 利用ログ：30日以内に削除
   - AI学習済みモデルに組み込まれたデータ：復元不可能なため、学習時点で完全除外のプロセスは定めていません
   - 決済情報：法令要件により7年保存（個人を特定できない形で匿名化）

2. **完全削除までの期間**：最大90日間

### 5.3 Xアクセス権限の取消し

1. ユーザーがX側の設定からShieldMeへのアクセス権限を取り消した場合：
   - 新たなデータ取得は停止します
   - 既に取得されたデータは、上記の保存期間に従って管理されます

## 第6条 ユーザーの権利

### 6.1 個人情報へのアクセス・開示請求

個人情報保護法に基づき、ユーザーは以下の権利を行使できます：

1. **情報開示請求**：当社が保有する自分の個人情報の内容を開示請求できます
2. **訂正請求**：不正確な情報の訂正を請求できます
3. **削除請求**：個人情報の削除を請求できます（ただし、法令要件により保存が必要な場合は除く）
4. **利用停止請求**：個人情報の利用停止を請求できます

### 6.2 請求方法

以下のメールアドレスから、「個人情報開示請求」と明記の上ご連絡ください：

- **メール:** support@ai-unmanned.com（準備中）
- **内容**：請求内容、ユーザーID、メールアドレス

当社は原則として**30日以内**に応答します。

### 6.3 同意の撤回

ユーザーがAI分析（カウンセリング機能等）の利用に同意した場合、いつでも同意を撤回できます。撤回後、当社はそれ以上のAI処理を行いません。

### 7.1 実施されるセキュリティ対策

1. **通信の暗号化**

3. **定期監査**
   - セキュリティ監査の定期実施（予定）
   - 脆弱性スキャンの実施

### 7.2 セキュリティインシデント時

1. データ漏洩等のセキュリティインシデントが発生した場合、当社は以下の対応を行います：
   - ユーザーへの通知（重大事件の場合は個人情報保護委員会に報告）
   - 原因調査・再発防止策の実施

## 第8条 GDPR対応（国際利用ユーザー向け）

### 8.1 GDPRとは

GDPR（一般データ保護規則）は、EUおよび欧経済領域（EEA）の居住者の個人情報を保護する規則です。

### 8.2 現状

ShieldMeは、**現時点ではGDPR完全対応を想定していません。** 以下の理由からです：

- EU・EEA居住者の本サービス利用を想定していない
- GDPR適用には国際法務の専門家相談が必須

### 8.3 今後の対応

EU・EEA居住者の利用が確認された場合、GDPR対応のための以下の施策を検討します：

- 同意管理ツール（Cookie同意）の導入
- DPA（Data Processing Agreement）の締結
- 標準契約条項（SCC）の適用
- 個人情報保護影響評価（DPIA）の実施
