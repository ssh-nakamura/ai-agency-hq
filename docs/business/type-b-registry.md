> 更新者: CEO
> 最終更新: 2026-02-26

---

## Type Bとは

市場調査に基づくニッチ独立ブランド展開。AIは生産手段として活用し、「仮想機関AI計画」の名前やAI制作であることは一切公開しない。

**エージェント分離ではなくタスク指示で切り替える。** 既存エージェント（writer / x-manager / video-creator等）にCEOが「Type Bタスク」として指示することで、同じエージェントがType A/Bを使い分ける。

---

## 事業立ち上げワークフロー

1. **analyst**: ニッチ市場調査（`/trend-research` + `/trend-scout`）
2. **CEO**: Go/NoGo判断（GTMスコア20点以上 → Go）
3. **CEO**: 本台帳のアクティブ事業一覧にエントリ追加
4. **CEO**: `content/type-b/{id}/` ディレクトリ作成
5. **CEO**: `content/type-b/{id}/brand.md` ブランドシート作成
6. **CEO**: 既存エージェントにType Bタスクとして指示（テンプレート下記）
7. **analyst**: 月次収益追跡
8. **撤退判断**: 3ヶ月連続赤字 → CEO判断で撤退

## 撤退基準

- **3ヶ月連続赤字**: CEOが撤退判断。コンテンツ資産は保存、運用停止
- **市場環境の急変**: 規制変更、プラットフォーム仕様変更等 → CEO即断
- **ブランドリスク**: 炎上・法的問題 → legal確認後、CEO判断で即撤退

## Type Bタスク指示テンプレート

CEOが既存エージェントにType Bタスクを指示する際に使用:

    【Type Bタスク】
    ブランドID: {id}
    ブランドシート: content/type-b/{id}/brand.md

    上記ブランドシートに従って制作してください。
    - 仮想機関AI計画の名前を出さない
    - AI制作であることを出さない
    - ブランドのトーン&マナーに従う

    タスク内容:
    {具体的な作業指示}
