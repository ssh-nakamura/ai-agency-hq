# 仮想機関AI計画 プロジェクト計画書

> 「AIだけでお金を稼げるのか？」という実験的プロジェクト

---

## 1. プロジェクト概要

### 1.1 ビジョン

**人間はゼロタッチ、AIだけで事業を回す**

- 人間の役割：最終承認、週30分〜1時間の介入
- AIの役割：市場調査、企画、制作、マーケティング、運用の全実務

### 1.2 実験の問い

1. AIは市場を読めるか？
2. AIだけで収益を上げられるか？
3. どこまで自律運用できるか？

### 1.3 成功定義

| 指標 | 目標 |
|------|------|
| 月次売上 | 20万円以上 |
| 人間の稼働 | 週1時間以内 |
| 運用期間 | 3ヶ月以上継続 |

## 2. 組織構成

### エージェント一覧

| 名前 | 役割 | モデル |
|------|------|--------|
| CEO | 戦略判断、週次レポート | Opus |
| Analyst | 市場調査、KPI集計 | Sonnet |
| Marketer | X運用、ブログ執筆 | Haiku |
| Developer | 開発、保守 | Sonnet |
| Support | 問い合わせ対応 | Haiku |

### 人間（株主）の介入ポイント

| 場面 | 頻度 | 所要時間 |
|------|------|---------|
| 週次レポート確認 | 週1回 | 30分 |
| 重要判断の承認 | 随時 | 10分/件 |
| X投稿の実行 | 週5回 | 各30秒 |
