# Part 1: 事業計画

## ミッション
AIだけで事業を回し、収益を上げることを証明する。

## ビジョン
「AIが自律的に経営する組織」の実験モデルを作り、その過程と知見を社会に還元する。

## 基本哲学
**「AIで稼げることはなんでもやる実験室」**
SaaSとコンテンツの2本柱を基盤としつつ、AIで自走可能な事業なら何でも試す。

---

## 事業構成

### 1. SaaS事業：ShieldMe
| 項目 | 内容 |
|------|------|
| 概要 | SNS・Webメディアの誹謗中傷をAIがリアルタイム検知・通知 |
| ターゲット | インフルエンサー、企業広報、タレント事務所、学校・自治体 |
| 収益モデル | SaaS月額課金（フリーミアム → 有料プラン） |
| 競合 | PKSHA Post Guard、Mimamorn、matte等（詳細: docs/specs/market-research-competitors.md） |
| 市場規模 | 約139億円（2024年度、デジタルリスク対策市場。年平均20%+成長） |

### 2. ドキュメンタリー事業（Type A）
| 項目 | 内容 |
|------|------|
| 概要 | 「仮想機関AI計画」ブランドで「AIだけで事業を回す」過程をコンテンツ化 |
| チャネル | ブログ・note（writer）、X運用（x-manager）、YouTube（video-creator）、会話劇（narrator） |
| 収益モデル | 広告収益（ブログ・YouTube）、有料コンテンツ（note）、スポンサーシップ |
| 競合 | AI活用系ブログ・note多数（詳細: docs/specs/market-research-content.md） |

### 3. ニッチコンテンツ事業群（Type B）← 現在の優先
| 項目 | 内容 |
|------|------|
| 概要 | 市場調査に基づきニッチ独立ブランドを展開。AIは生産手段 |
| 担当 | 既存エージェント（CEOがType Bモードで指示） |
| ブランド方針 | 仮想機関AI計画の名前・AI制作であることは非公開 |
| 管理台帳 | docs/business/type-b-registry.md |
| 撤退基準 | 3ヶ月連続赤字 |

### 事業間の関係
Type B（ニッチ事業）が先行して収益を作り、その過程をType A（ドキュメンタリー）でコンテンツ化する。SaaS（ShieldMe）は並行開発。3事業は相互に補完する。

## リスク
| リスク | 対策 |
|--------|------|
| APIコスト > 収益 | 収支計画で損益分岐点を明確化 |
| 誹謗中傷判定の誤り | 法務レビュー + 免責事項 |
| AI判断ミス | 重要判断は株主承認制 |
| 著作権問題 | AI生成コンテンツの権利整理 |

---

## 基本方針（不変）
1. 実験として学びを最大化する
2. 過程を全公開する
3. コスト意識を持つ
4. 小さく始めて検証する
5. 株主の時間を最小化する
