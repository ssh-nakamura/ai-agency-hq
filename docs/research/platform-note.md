## サマリー（3行）

noteは2025年11月期に売上高41.41億円（前期比+25%）、四半期利益初の1億円突破と過去最高を更新し、メンバーシップ売上は前年比+81.3%で急成長中。売れているのは「AI活用」「SNS運用」「育児」の3ジャンルだが、「職種特化型AI活用」「AI業務自動化テンプレート集」「日本語AIニュースレター（Substack型）」はnoteで供給が不足しており、Trend Arbitrageの3条件（Breakout + Information Gap + High Intent）を満たす空白地帯である。有料記事よりメンバーシップの方が成長率が3倍速く（+81.3% vs +26.8%）、月額500円のエントリープランから始めるサブスク型が最も安定した収益構造を生む。

## Part 2: noteで今実際に売れているもの

### 2-1. ジャンル別の実態

note公式が2026年1月に約30万件の有料記事（2024年12月〜2025年10月投稿分）を分析し、14カテゴリに分類した結果、以下のジャンルが急成長している。

#### 急成長ジャンル（Breakout）

| ジャンル | 成長の特徴 | 新規参入の容易さ | 代表的な収益モデル |
|---------|-----------|----------------|------------------|
| AI活用・テクノロジー | 2026年で最も勢いのあるジャンル | 高（職種特化で差別化可能） | 有料記事（¥1,500〜¥3,000）+ メンバーシップ |
| SNS運用 | インスタ・X運用ノウハウの需要増 | 中（実績提示が求められる） | 有料記事 + コンサル誘導 |
| 育児・子育て | メンバーシップTop10のうち7人が1年以内の新規 | 高（体験談ベースで参入容易） | メンバーシップ（月額500円〜） |
| 副業・在宅ワーク | 常に一定の需要あり | 中（競合多い、差別化必要） | 有料記事（¥500〜¥3,000） |
| ビジネス・マーケティング | BtoB寄りの専門コンテンツ | 中（専門性が差別化要因） | メンバーシップ + 有料記事 |

**出典**: [note公式 30万件有料記事分析（2026年1月）](https://prtimes.jp/main/html/rd/p/000000360.000017890.html)、[manamina](https://manamina.valuesccg.com/articles/4766)

#### 育児カテゴリの特筆すべき事実

育児カテゴリのメンバーシップ売上トップ10のうち**7人が1年以内に新規参入したクリエイター**。これはnote全カテゴリの中で最も新規参入者が上位を占めやすいカテゴリであることを意味する。

**出典**: [note公式プレスリリース](https://prtimes.jp/main/html/rd/p/000000360.000017890.html)

### 2-2. 価格帯別の売上実態

| 価格帯 | コンテンツの種類 | 平均価格 | 備考 |
|--------|----------------|---------|------|
| ¥500以下 | お試し・入門記事 | --- | ファーストキャッシュ向け |
| ¥500〜¥1,500 | 最も売れやすい価格帯 | --- | 心理的に「試してみよう」と感じる範囲 |
| ¥1,000〜¥2,000 | 実用ノウハウ系の中心 | ¥1,842 | note公式分析（上位20%の平均） |
| ¥500〜¥1,000 | 読み物・体験談系 | ¥983 | 同上 |

### 2-3. トップクリエイターの収益規模

| 指標 | 値 | 出典 |
|------|-----|------|
| 上位1,000クリエイターの平均年収 | ¥11,000,000〜¥15,150,000 | note公式（発表時期により変動） |
| 累計100万円以上売上クリエイターの割合 | 全体の約0.2% | note有料記事事例研究 |
| 有料記事販売クリエイター数 | 155,000人以上 | note公式 |

**出典**: [note株式会社 公式発表](https://note.jp/n/n131d989e370e)、[GENSEKIマガジン](https://magazine.genseki.me/entry/sp_saito_20231031)

**So What?**: 上位1,000人の平均年収は1,000万円超だが、全体の0.2%に過ぎない。つまり155,000人中の1,000人＝上位0.6%が高収益を独占。残りの99.4%の平均月収は数万円以下と推定される。「noteで稼げる」は事実だが、極めて偏った分布であることを認識すべき。

### 3-3. メンバーシップの価格設計（note公式3,000件分析の知見）

note編集部が約3,000件のメンバーシップを分析した結果:

| 分析項目 | 結果 |
|---------|------|
| 最も多い月額価格 | ¥500 |
| 売上が高いメンバーシップのプラン数 | 2〜4つ |
| 各プラン間の価格差 | 約3倍（例: ¥500 / ¥1,500 / ¥5,000） |
| 売上が高いメンバーシップの共通点 | 「何が得られるか」が簡潔・具体的に説明されている |
| 効果的な訴求手法 | 日常的なものの価格（「コーヒー1杯分」等）にたとえる |

**出典**: [note編集部 3,000件メンバーシップ分析](https://note.com/notemag/n/n65bfc3cf1d88)

**推奨プラン設計**: ¥500（エントリー）→ ¥1,500（標準）→ ¥5,000（プレミアム）の3段階。エントリーで集客し、価値を実感した層を上位プランへ誘導。

## Part 5: 空白地帯の特定（Trend Arbitrage分析）

### 5-1. Trend Arbitrageフレームワーク適用

3条件全てを満たす空白地帯を特定する:
1. **Breakout（急上昇）**: 検索・需要が急伸
2. **Information Gap（供給不足）**: 良い有料記事が少ない
3. **High Intent（収益性）**: 購入意欲が高い、メンバーシップに繋がる

### 5-2. 空白地帯候補一覧

#### 空白地帯 S-1: 職種特化型AI業務効率化テンプレート集

| 評価項目 | 評価 |
|---------|------|
| Breakout | ○: AI活用はnoteで最も成長の速いジャンル |
| Information Gap | ○: 「営業職向けAI活用」「経理向けAI活用」等、職種特化型は供給不足 |
| High Intent | ○: 「作業時間50%削減」等の具体的成果が提示可能で課金意欲が高い |
| **総合評価** | **S（最有力）** |

- **想定価格**: ¥1,500〜¥3,000/記事、メンバーシップ¥500〜¥1,500/月
- **想定ターゲット**: 30〜50代会社員、業務効率化に関心がある層
- **コンテンツ例**: 「経理担当者のためのChatGPTプロンプト50選」「営業職が明日から使えるAI議事録術」
- **参入の根拠**: noteで「AI活用」は急成長中だが、対象が曖昧な汎用記事が多い。職種ごとに特化した記事は少なく、Gumroadのテンプレート販売モデルをnoteに持ち込む余地がある

#### 空白地帯 S-2: 日本語AIニュースレター（Substack型の定期キュレーション）

| 評価項目 | 評価 |
|---------|------|
| Breakout | ○: 英語圏ではThe Rundown AI（200万読者）等が確立、日本では空白 |
| Information Gap | ○: 日本語で質の高いAI動向の定期配信がほぼ存在しない |
| High Intent | ○: Substackで$1M+/年のAIニュースレターが複数存在。日本での需要は推定で2〜4年遅れ |
| **総合評価** | **S（最有力）** |

- **想定価格**: メンバーシップ¥500/月（エントリー）、¥1,500/月（詳細分析付き）
- **想定ターゲット**: AIに関心のあるビジネスパーソン、エンジニア、経営者
- **コンテンツ例**: 「今週のAIニュース10選（日本語解説付き）」「海外AI事例を日本企業にどう活かすか」
- **参入の根拠**: 英語圏のSubstackで確立済みだが日本語版が不在。noteのメンバーシップ機能でニュースレター型は再現可能。英語圏→日本のコンテンツマネタイズ手法のラグは2〜4年と推定（自社調査: `docs/research/overseas-niche-analysis.md`）

#### 空白地帯 A-1: 育児×テクノロジー（AI活用した子育てハック）

| 評価項目 | 評価 |
|---------|------|
| Breakout | ○: 育児カテゴリはメンバーシップで新規参入者がTop10の7割を占める |
| Information Gap | △: 育児記事は多いが「育児×AI/テクノロジー」の掛け合わせは少ない |
| High Intent | ○: 育児メンバーシップの課金意欲は高い（新規7/10がTop10入り） |
| **総合評価** | **A（有力）** |

- **想定価格**: メンバーシップ¥500/月
- **コンテンツ例**: 「ChatGPTで子どもの宿題を一緒にやる方法」「AIで時短する離乳食レシピ生成」

#### 空白地帯 A-2: AI×特定資格取得ハウツー

| 評価項目 | 評価 |
|---------|------|
| Breakout | △: 資格取得は安定需要だが急上昇ではない |
| Information Gap | ○: マイナー国家資格のAI活用学習法は供給ゼロに近い |
| High Intent | ○: 資格取得者の課金意欲は高い。通信講座アフィリエイトも高単価 |
| **総合評価** | **A（有力）** |

- **想定価格**: ¥1,000〜¥2,000/記事
- **コンテンツ例**: 「宅建試験にAIで3ヶ月合格した方法」「FP2級の頻出問題をChatGPTで効率学習」

#### 空白地帯 B-1: Notionテンプレート × note販売

| 評価項目 | 評価 |
|---------|------|
| Breakout | △: Gumroadでは確立済みだが、noteでのNotionテンプレート販売は未発達 |
| Information Gap | ○: noteでNotionテンプレートを体系的に販売しているクリエイターは少ない |
| High Intent | △: Gumroadの主力だが、noteユーザーの購買行動との相性は要検証 |
| **総合評価** | **B（次点）** |

#### 空白地帯 B-2: 介護×テクノロジー

| 評価項目 | 評価 |
|---------|------|
| Breakout | △: 市場は拡大中だが、noteでの需要は未知数 |
| Information Gap | ○: 介護テクノロジーの実践ガイドはnoteでほぼ不在 |

### 6-2. 推奨するnote収益モデル

```
[収益構造の設計]

  → 集客・ブランド構築の入口

  → 安定収益の柱
  → エントリー¥500で集客 → 標準¥1,500へ誘導
  → 特典: 限定記事、掲示板、メール配信

[月収¥67,500達成シミュレーション]
  パターンA: メンバーシップ¥500×100人 + 有料記事¥1,500×12本/月 = ¥50,000 + ¥18,000 = ¥68,000
  パターンB: メンバーシップ¥1,500×30人 + 有料記事¥2,000×6本/月 = ¥45,000 + ¥12,000 = ¥57,000
  パターンC: メンバーシップ¥500×50人 + ¥1,500×20人 = ¥25,000 + ¥30,000 + 有料記事¥1,500×10本 = ¥70,000

  ※上記は売上ベース。手取りは約85.5%（クレカ決済の場合）
  ※パターンAの手取り: ¥68,000 × 0.855 = ¥58,140
  ※目標¥67,500達成には売上ベースで約¥78,950必要（¥67,500 / 0.855）
```

### 6-3. 匿名運用の注意点

| 項目 | 内容 |
|------|------|
| noteの匿名運用 | 技術的に可能。ペンネーム・ペルソナでの運用に制限なし |
| 実名の必要性 | 特定商取引法の表記義務はnote上の有料記事販売では**note社が販売事業者となるため不要** |
| 複数アカウント | note利用規約上、1人で複数のアカウント作成は原則禁止 |
| ペルソナの信頼性 | 匿名でも「○○業界10年」等のペルソナ設計で信頼は構築可能。週1更新×6ヶ月でフォロワー3,000人増の事例あり |
| 段階的な信頼構築 | 初期は¥500で販売 → 100件売れたら¥800に値上げ → さらに¥1,200へ。トータル¥130,000超の事例あり |

**注意**: 1人で複数のType Bブランドを展開する場合、noteの1アカウント制限を考慮する必要がある。メンバーシップのプラン分けで対応するか、異なるメールアドレスでの運用（規約違反リスクあり）を検討する必要がある。この点はCEO判断を仰ぐべき。

## 示唆（So What?）

### 1. noteの成長は「メンバーシップ」が牽引している
有料記事の成長率+26.8%に対し、メンバーシップは+81.3%。サブスク型が明確に勝っている。新規参入するならメンバーシップ型を主軸にすべき。

### 2. 「新規参入者が勝てる」構造が現在のnoteにはある
育児カテゴリのメンバーシップTop10のうち7人が1年以内の新規参入者。プラットフォームの成長期（GMV四半期ごとに加速）に乗ることで、後発でも上位を取れる環境がある。

### 3. AI活用ジャンルは「汎用→職種特化」への分化フェーズ
「AI活用」は最も成長の速いジャンルだが、汎用的な記事は飽和しつつある。Gumroadで確立された「テンプレート販売モデル」をnoteに持ち込み、職種ごとに特化した記事を出すことで差別化が可能。

### 4. Substackの「AIニュースレター」モデルは日本で空白
英語圏ではThe Rundown AI（200万読者）等がAIニュースレターで7桁$/年を達成しているが、日本語で同等の品質を持つ定期配信はnoteに存在しない。英語圏→日本のラグ（推定2〜4年）を考えれば、今が参入のタイミング。

### 5. 文字数は売上に影響しない
note公式の30万件分析で、文字数と売上の相関係数はほぼゼロ。「長く書けば売れる」は誤り。短くても購入者の具体的な課題を解決するコンテンツが勝つ。

### 6. 複数ブランド展開にはnoteの1アカウント制限がボトルネック
Type B事業で複数ニッチを展開する場合、noteの複数アカウント禁止規約が制約になる。1アカウント内でメンバーシップのプランを分けるか、プラットフォームを分散（note + ブログ + Kindle）するかの戦略判断が必要。

## CEOへの推奨アクション

1. **最優先**: S評価の2候補（職種特化型AI業務効率化テンプレート集 / 日本語AIニュースレター）のうち、どちらをパイロットとするかCEO判断を仰ぐ。どちらも初期投資¥0で開始可能。
2. **メンバーシップ型を主軸にせよ**: 有料記事（+26.8%）よりメンバーシップ（+81.3%）の方が3倍速い成長率。月額¥500エントリー → ¥1,500標準の2段階プランで開始を推奨。
3. **KPI目標の再計算**: 手数料を考慮した月収¥67,500達成には、売上ベースで約¥78,950が必要（手取り率85.5%）。この数字を`docs/status.md`のKPI目標に反映すべき。
4. **複数ブランド展開の制約調査**: noteの1アカウント制限について、legal部門にリスクレビューを依頼すべき。
5. **Substack日本語対応の監視**: Substackが日本語対応を進めている。noteの独占状態が崩れる可能性があるため、月次で動向を監視すべき。

## リスク・不確実性

| リスク | 内容 | 優先度 |
|--------|------|--------|
| 推定月収の不確実性 | 月収の具体的数値はクリエイター個人の報告に依存。過大報告の可能性あり | 高 |
| noteの規約変更 | AI生成コンテンツへの制限が入る可能性（現在は原則許可） | 中 |
| メンバーシップの解約率 | 解約率のデータが非公開。月額¥500でも解約率が高ければ安定しない | 中 |
| 上位0.2%問題 | 累計100万円以上売上は全体の0.2%。大半のクリエイターは月数万円以下と推定 | 高 |
| Substack日本上陸 | 日本語対応が本格化すれば、noteの独占が崩れる可能性 | 低（中期的にはリスク上昇） |
| 複数アカウント制限 | Type B事業の複数ブランド展開の制約となる | 中 |
| 有料コンテンツ比率24〜25%の出典 | 複数メディアで引用されているが、note公式の一次ソースの特定ができなかった | 低 |

**データの限界**:
- note公式の30万件分析はカテゴリ別の売上金額・成長率の詳細な数値を開示していない。「急成長」の定義は定性的記述にとどまる。
- トップクリエイターの平均年収は発表時期によって¥11,000,000〜¥15,150,000と幅がある。直近の公式数値を特定できなかった。
- 空白地帯の「推定月収」は、類似ジャンルの実績と市場規模からの推計であり、保証値ではない。
