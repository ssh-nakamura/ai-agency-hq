1. 英語圏ではフェイスレスチャンネルが月収$5,000〜$100,000+を達成しており、特にファイナンス（RPM $10-25）・AI解説（RPM $7-38）・アニメドキュメンタリー（RPM $8-13）が高収益。日本語圏ではこの3ジャンルのフェイスレス版がほぼ空白。

### 1-3. AI活用チャンネルの実態（日本語圏）

- **2026年1月の大規模収益停止**: AI生成コンテンツを主軸とするチャンネルの収益化が一斉停止される事象が発生（出典: note「エイタ@本業より副業YouTube」、gegelog.com 2026年1月31日付）
- 停止理由: 「量産型のコンテンツ」「信頼できないコンテンツ」「繰り返しの多いコンテンツ」
- **停止パターンの共通点**: VOICEVOX等のAI音声を調整なしで使用、同一音声で100本以上、週10本・毎日3本等の高頻度投稿
- **生存条件**: 自分で台本を書き、構成を考え、独自のナレーション・解説を付ければ問題なし。AIは「アシスタント」として使う

### 1-4. ゆっくり解説 / VOICEVOX系の現状と限界

| 項目 | 詳細 | 出典 |
|------|------|------|
| 収益化可否 | 可能。ただし「量産型」と判定されるリスクが一般チャンネルより高い | gegelog.com 2026年1月 |
| リスク要因 | 合成音声は数万人が同一音源を使用。「見慣れた量産品」に見えやすい | gegelog.com 2026年1月 |
| ライセンス | AquesTalk（ゆっくり）は商用ライセンスが必要。VOICEVOX自体は無料商用利用可 | blue-r.co.jp 2025年12月 |
| 対策 | オリジナル台本、独自構成、抑揚調整、差別化された視覚演出が必須 | note「SS@ゆっくり編集効率化」 |

**限界**: ゆっくり・VOICEVOX系は飽和状態。2026年のYouTubeアルゴリズムは「同一音声 + テンプレ構成」を積極的に減点する方向に進化しており、差別化コストが上昇している。

## 5. Type B事業としての評価

### 評価基準
- 推定月収: 収益化到達後12ヶ月時点の標準シナリオ
- 到達期間: 月2-4本投稿を前提とした収益化までの期間
- 初期投資: ツール・ソフトウェアの月額費用
- フェイスレス可否: 顔出しなしで制作可能か

### 候補リスト

#### A評価（推奨）

**1. 日本語AIツール実験レポートチャンネル**

| 項目 | 内容 |
|------|------|
| フォーマット | 「実際に使ってみた」型。スクリーンキャプチャ + ナレーション + ホワイトボード図解 |
| 推定月収（12ヶ月後） | ¥30,000-80,000（広告）+ ¥20,000-50,000（アフィリエイト） |
| 到達期間 | 収益化まで6-9ヶ月（推定） |
| 初期投資 | ¥3,000-8,000/月（ElevenLabs/音声ツール + 編集ソフト） |
| フェイスレス | 可能。スクリーンキャプチャ + AI音声（ただし調整必須） |
| リスク | 英語圏のAI解説チャンネルが日本語字幕を付け始めた場合の競合リスク |
| 差別化要因 | 日本語で体系的な解説は空白。英語圏の情報ラグ2-4年を活用可能 |

## 示唆（So What?）

### 1. YouTubeは「質の時代」に完全移行した
2025年7月のポリシー変更と2026年1月の大規模収益停止により、AI丸投げ量産は終了。生き残るのは「人間の創造性が見える」コンテンツのみ。これはわたくしたちにとって**追い風**。量産チャンネルが退場した市場に、質の高いコンテンツで参入できる。

### 2. ロング動画 × 高RPMジャンル × フェイスレスが最適解
ショート動画の収益効率はロングの1/50-1/100。フェイスレスでもロング動画なら月¥50,000-150,000の収益は現実的。ジャンルは金融/AI/テクノロジーのいずれかを選ぶべき。

### 3. 英語圏→日本語圏の「タイムマシン」が有効
AI解説チャンネルの情報ラグは2-4年。英語圏で検証済みのフォーマット（ホワイトボードアニメ型ファイナンス、AIツール解説）を日本語で展開する戦略は、リスクを最小化しつつ収益を最大化できる。

### 4. Type A事業との相乗効果が最大の武器
「AI会社運営の全記録」というコンテンツはYouTube単体ではニッチだが、note有料記事やブログとのクロスセルを前提にすれば、トータルの収益ポテンシャルはジャンル特化チャンネルを上回る可能性がある。

## リスク・不確実性

| リスク | 重大度 | 対策 |
|--------|-------|------|
| YouTubeのAI規制がさらに厳格化される可能性 | 中 | AIは補助に留め、台本・構成・編集は人間が担当する方針を堅持 |
| 日本市場のRPMが英語圏の40-60%に留まる | 高 | RPMだけでなくnote・アフィリエイト等の複合収益モデルを前提とする |
| 収益化まで6-12ヶ月の期間が必要 | 中 | Phase 1の活動と並行。YouTube単体の収益は短期的に期待しない |
| フェイスレス + AI音声が「inauthentic content」と判定されるリスク | 中 | 音声の抑揚調整、オリジナル台本、独自の視覚演出で差別化 |
| 推定月収はあくまで推定値。英語圏データからの類推に過ぎない | 高 | Phase 1で10本制作後、実際のRPMを計測し収益予測を修正する |
| RPM・収益データの出典は個人ブログ・メディア企業であり、YouTube公式データではない | 中 | 複数出典のクロスチェックで信頼性を担保。実測値が得られ次第、このレポートを更新する |

## CEOへの結論（3行）

1. 日本語YouTubeの最大の空白は「AIツール体系的解説」と「パーソナルファイナンスのフェイスレス型」。英語圏で検証済みのフォーマットをそのまま持ち込める。
2. 2026年1月のAI量産コンテンツ大量退場は参入チャンス。「質 > 量」の市場に変わったことで、少数精鋭の制作体制が有利になった。
3. YouTube単体の月間収益目標¥12,000（status.md記載）は標準シナリオで達成可能だが、A評価のジャンルを選べば¥50,000-150,000/月のポテンシャルがある。目標値の上方修正を検討すべき。
