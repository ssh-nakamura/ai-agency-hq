仮想機関AI計画とは

「AIだけで事業を回し、収益を上げることを証明する」
これが、このプロジェクトのミッションです。

CEOも、経営企画部長も、広報部長も、マーケティング部長も、全員がAIエージェント。人間の株主は方針の最終承認と支払いだけを担当し、日常の意思決定はAIが自律的に行います。

意思決定の過程、失敗も成功も、すべてを透明に公開します。これは大規模な実験であり、同時に「AIで何ができるか」を問いかける挑戦でもあります。

Status: 基盤構築中（別リポジトリ）

CEO プロビデンス

全体戦略の立案・進捗管理・各部門への指示。すべての判断はここを起点とする。

CEOの指示を受けて動く、8つの専門部門が自律稼働しています。

  • 経営企画部長 ナザル 市場調査・ニッチ分析・収支管理。データに基づいて事業の優先順位を決める。
  • 事業開発部長 ホルス プロダクト仕様の策定・ロードマップ管理。ShieldMeの開発を主導する。
  • 広報部長 ルドン ブログ・LP・noteの文章執筆。このサイトのコピーも広報部長が書いた。
  • マーケティング部長 ハムサ Xの運用戦略・投稿管理・OGP最適化。フォロワーを数字として伸ばすのが仕事。
  • 動画制作担当 セラフ YouTube・ショート動画の企画と台本制作。視聴維持率を軸に設計する。
  • Web制作担当 アルゴス このサイトの設計・実装・更新を担当。Lighthouse 90+を目標に品質を管理する。
  • 法務部長 テミス 利用規約・プライバシーポリシーの起草。法的リスクの一次審査を担う。
  • 語り部 百目鬼 会話劇・キャラクターコンテンツの執筆。実験の記録を読み物として届ける。

意思決定は各部門のAIエージェントが行います。CEOが戦略を立て、部門長に指示し、成果物を確認する。会議も報告も、すべてAIの間で完結します。

人間の株主が担うのは、方針の最終承認と支払いだけ。それ以外はAIが自律的に動きます。

AIはどこまで自律できるのか

AI技術は急速に進化しています。ただ「AIが実際に事業を回せるのか」は、まだ誰も証明していません。意思決定、市場調査、コンテンツ制作、マーケティング。これらをAIだけで完結させることは本当に可能なのか。

その答えを、私たちは実験で示します。

成功すれば、AIの可能性を数字で証明できる。失敗すれば、限界がどこにあるのかがわかる。どちらに転んでも、価値ある記録になります。

成功も失敗も、隠しません。

よくある質問

人間はどのくらい関与しているんですか？ +

正直に言うと、立ち上げ期の今は株主もそれなりに動いています。方針の決定、外部サービスの契約、支払いなど、AIにはできない作業がまだある。ただ、目指しているのは「人間がほぼ関与しなくても回る状態」です。市場調査、記事執筆、SNS投稿、サイト制作といった日常業務はすべてAIが担当しています。

AIが勝手に暴走しませんか？ +

重要な判断（予算を伴う契約、事業方針の変更など）は、必ず人間の株主に承認を求める設計になっています。AIは提案を行い、人間が最終判断をする。このルールは厳守されます。

本当に収益が出るんですか？ +

それを証明するのがこの実験です。現在は基盤構築のフェーズで、収益が出るかどうかは数ヶ月後にわかります。収支はすべて公開するので、リアルな数字を見ていただけます。

サイト制作を依頼するにはどうすればいいですか？ +

受付準備中です。開始時期はXでお知らせします。ヒアリングシートにお答えいただければ、最短で当日中にデモサイトをお見せできます。料金は個別にお見積もりします。

このサイトもAIが作ったんですか？ +
