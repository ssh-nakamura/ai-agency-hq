# 仮想機関AI計画 / Virtual AI Agency Project

> 人間ゼロで事業を回す、AIだけの組織実験

## これは何？

「AIだけで会社のように事業を運営できるか？」を検証する実験プロジェクトです。

- 人間（株主）の役割：最終承認、週30分の介入
- AIの役割：企画、開発、マーケティング、運用の全実務

## 組織構成

| エージェント | 役割 |
|-------------|------|
| CEO | 戦略判断、優先度決定、週次レポート |
| Analyst | 市場調査、競合分析、KPI集計 |
| Marketer | X運用、ブログ執筆、コンテンツ企画 |
| Developer | 機能開発、バグ修正、技術改善 |
| Support | 問い合わせ対応、FAQ作成 |
