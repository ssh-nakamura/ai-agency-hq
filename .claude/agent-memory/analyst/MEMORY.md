## 重要な学び

### 市場データの扱い方
- 日本のデジタルリスク市場: 2024年度139億円（MM総研）、年20%成長
- 検知系サービス: 35.3億円、対策系: 42.1億円（2021年度）
- 既存サービスは月額5万円〜が主流。個人向け月額1万円未満は**空白地帯**

### コンテンツ事業の現実（2026年）
- note有料記事の平均価格: 実用系1,842円、読み物系983円
- ブログAdSense RPMは低下傾向: 2018年400〜700円 → 2026年250〜350円
- AI活用ジャンルはnoteで急成長（メンバーシップ+81.3% YoY）
- **示唆**: note有料記事を主軸、ブログは補完的位置づけが現実的

### YouTube収益化の壁（2026年）
- 収益化条件: 登録1,000人 + 総再生4,000時間（または90日ショート300万回）
- ショート動画RPM: 0.003〜0.01円/再生（100万再生≒3,000〜10,000円）
- 2026年アルゴリズムはAI生成大量投稿を評価減（質重視）
- **示唆**: 量産ではなく、質の高い「実験レポート動画」に特化すべき

### KPI設計の原則
- 3シナリオ（楽観・標準・悲観）で設定し、標準を意思決定のベースラインとする
- 標準シナリオ達成率80%以上 = 順調、50%未満 = 危険
- 3週連続未達 = 構造的問題、戦略見直しを提案

### コスト構造の要注意点
- X API Basic（月¥15,000）はPhase 1で大きな負担。Phase 1開始前に株主承認必須
- ShieldMeのAPI単価は**仮定**。Phase 1で100件実測し、価格モデル再検証必須
- Claude Max契約（月¥30,000想定）の実額を株主に確認し、`finances.md`に記録する

## データソースの記録

### 信頼できる出典
- MM総研: デジタルリスク市場規模（公式プレスリリース）
- note公式: 有料記事30万件分析（2026年1月発表）
- YouTube公式: 収益化条件、パートナープログラム規約
- Google AdSense公式ヘルプ: RPMの定義と計算方法

### 注意が必要な情報源
- 個人ブログのRPM報告: サイトのジャンル・質によって大きく変動
- YouTube収益の「1再生あたり○円」: チャンネルによって桁違いの差がある
- SaaS平均価格: BtoB中心のデータが多く、BtoC個人向けは別物

## KPI管理の実務

### 週次ルーティン
1. 毎週金曜: `docs/state.json` のKPI更新
2. 前週比の変動分析（±20%以上の変化は原因を調査）
3. 異常値の検知（目標の50%未満は即CEOに報告）
4. CEOの週次レポート用に数字サマリーを提供

### 月次ルーティン
1. `docs/finances.md` の収支記録更新
2. 実際のコストと予算の乖離チェック
3. 損益分岐点の再計算（コスト変動時）
4. 市場動向の変化チェック（競合の新動向等）

## 財務管理（CFO兼務）

### 収支台帳の原則
- 実績ベースで記録（見積もりではなく、発生した金額のみ）
- 新規費用が発生したら即座にCEOに報告
- 月額¥30,000以上の支出は株主承認必須

### コスト分類
- 固定費: Claude Max、ドメイン
- 変動費: X API、ホスティング、外部API
- 投資: 開発外注、ツール導入（都度承認）
