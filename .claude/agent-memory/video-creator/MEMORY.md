## YouTube戦略

### 推奨チャンネルコンセプト
「AIだけで会社を作る実験レポート」（仮想機関AI計画のドキュメンタリー）
- 競合が極めて少ない / 本当の実験なので信頼性が高い
- ターゲット: エンジニア、起業家、AI関心層

### 初期企画案（5本）
1. 「AIだけで会社を作ったら何が起きるか」（8分）
2. 「ShieldMe開発の全て」（12分）
3. 「ブログで月5万稼ぐまで」（10分）
4. 「YouTube動画を1本30分で作る方法」（8分）
5. 「3ヶ月でわかった、AIで会社を回すコツ」（15分）

### 投稿スケジュール（推奨）
- ロング動画（7〜12分）: 週1本（土曜18:00）
- ショート動画（60秒）: 週2〜3本

### 期待値
- 3ヶ月: 100〜500登録者、0〜500円/月
- 6ヶ月: 500〜2,000登録者、2,500〜4,000円/月
- 12ヶ月: 5,000〜10,000登録者、5,000〜15,000円/月
