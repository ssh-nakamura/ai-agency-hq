## YouTube戦略

### 推奨チャンネルコンセプト

**テーマ**: 「AIだけで会社を作る実験レポート」（仮想機関AI計画のドキュメンタリー）

**理由**:
- 競合が極めて少ない
- 本当の実験なので信頼性が高い
- 成功・失敗両方が興味深い
- ブログ、note、書籍への展開が容易

**ターゲット**: エンジニア、起業家、AI関心層

### 初期企画案（5本）

1. **「AIだけで会社を作ったら何が起きるか」**（8分）
   - フック: 「AIだけで会社を作りました。月額¥8,000で。」
   - 狙い: チャンネルの世界観説明 × バズより質重視

2. **「ShieldMe開発の全て」**（12分）
   - プロダクト開発エピソード、信頼構築

3. **「ブログで月¥50,000稼ぐまで」**（10分）
   - 実績の数字化、「AIで稼ぐ」の信頼性

4. **「YouTube動画を1本30分で作る方法」**（8分）
   - 実用的なハウツー、高コンテンツ密度

5. **「3ヶ月でわかった、AIで会社を回すコツ」**（15分）
   - ここまでの学びまとめ、長尺で深堀り

### 投稿スケジュール（推奨）

- **ロング動画（7～12分）**: 週1本（土曜18:00）
- **ショート動画（60秒）**: 週2～3本（平日昼+夜+土曜）

### 期待値

- 3ヶ月: 100～500登録者、¥0～500/月
- 6ヶ月: 500～2,000登録者、¥2,500～4,000/月
- 12ヶ月: 5,000～10,000登録者、¥5,000～15,000/月
- 24ヶ月: 黒字化（¥50,000～200,000/月）

## YouTube SEO重要ポイント

### 2026年版アルゴリズム変化

- 「平均再生時間」より「視聴完了後の満足度」を重視
- ショート動画: 「瞬間満足度」と「連続視聴性」
- ロング動画: コメント欄の活動度が重視
- ショート + ロングの両立で総再生時間増加 → アルゴリズム評価向上

### チェックリスト（全動画必須）

```
タイトル（60字以内）:
  □ キーワード含む
  □ 数字を入れている（¥10,000, 3ヶ月など）
  □ パワーワード（「ついに」「明かします」）

説明文:
  □ 最初の2行に核心
  □ キーワード自然に含める（3回）
  □ リンク配置（ブログ、他動画、note）

サムネイル:
  □ テキスト 3～5語（大きく、可読性重視）
  □ コントラスト高い配色（赤×黄、青×白）
  □ 顔 or 大きな図解を含む

タグ:
  □ 主要キーワード 3～5個
  □ 長尾キーワード 5～10個

エンドスクリーン:
  □ 次の動画へ導線
  □ チャンネル登録ボタン
```
