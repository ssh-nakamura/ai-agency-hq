# 動画企画

## 引数
- $ARGUMENTS: テーマまたは指示（例: "AI会社の1日", "ShieldMe紹介動画"）
  - 引数なし → CEOがテーマを決めてから開始
  - `--shorts` → ショート動画（60秒以内）
  - `--type-b {brand-id}` → Type Bモード

## ワークフロー

### Step 1: 企画（CEO判断）
- テーマと切り口
- 形式（解説 / ドキュメンタリー / チュートリアル / ショート）
- Type A or Type B
- ターゲット視聴者
- 目標尺（5分 / 10分 / 60秒）

### Step 2: 台本作成（video-creator）
video-creatorをTask起動:

```
テーマ: {テーマ}
形式: {形式}
Type: A / B
目標尺: {分}
ターゲット: {視聴者}

以下を作れ:
1. タイトル案（3案、SEO意識）
2. サムネイル案（テキスト + 構図の説明）
3. 構成（イントロ→本編→CTA、タイムスタンプ付き）
4. 台本（セクションごとのナレーション文）
5. 必要素材リスト（画像、BGM、SE）

成果物: content/videos/{slug}-plan.md
```

### Step 3: レビュー（CEO）
- 視聴者が最初の10秒で離脱しない構成か
- タイトルとサムネイルでクリックしたくなるか
- 尺が適切か

## 出力
成果物: `content/videos/{slug}-plan.md`

## 注意
- 映像・音声の実制作はこのスキルのスコープ外
- 撮影・編集ツール契約は株主承認待ち（S-002）
