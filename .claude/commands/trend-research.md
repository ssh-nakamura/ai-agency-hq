# Trend Research — コンテンツ制作レベルのトレンド調査

> 「今何がバズっているか」をプラットフォーム横断で調査するスキル。
> `/trend-scout`（GTM戦略レベル）とは棲み分け: こちらはコンテンツ制作のネタ探し用。

## 起動

    /trend-research                    # インタラクティブモード
    /trend-research AI副業             # テーマ指定
    /trend-research 英語学習 tiktok    # テーマ + PF指定

CEOがwriter/x-manager/video-creatorをTask起動して実行する。引数がなければテーマ・対象PFをCEOに確認してから開始。

## 3層SNS検索アーキテクチャ

### 1層: Grok API MCP（メイン — X/Twitter検索）

**ステータス**: 株主承認待ち（S-004: xAI APIキー未取得）

利用可能になったら以下を実行:
- X/Twitterのリアルタイムバズ検索
- トレンドハッシュタグの取得
- バイラル投稿の発見
- インフルエンサーの発言追跡

**検索パターン**:

    {テーマ} バズ OR 話題 OR トレンド
    {テーマ} min_faves:100
    {テーマ} lang:ja since:{1週間前}
    {テーマ} lang:en viral OR trending

### 2層: Xpoz MCP（補完 — Instagram/TikTok/Reddit）

**ステータス**: 株主承認待ち（Grok APIキーと同時に設定予定）

Grok非対応プラットフォームをカバー:
- Instagram: ハッシュタグトレンド、リール人気
- TikTok: バイラル動画、トレンドサウンド
- Reddit: サブレディットの話題、コミュニティの熱量

**注意事項**:
- Free枠（100K件/月、$0）のみ使用。課金情報は登録しない
- トレンド調査のみに使用（社内情報を含む検索禁止）
- 不要になったら `claude mcp remove xpoz` で即座に無効化

**2層が利用不可の場合**: 3層（WebSearch）でカバー。スキルは2層なしでも動作する。

### 3層: WebSearch（フォールバック — 常時利用可能）

1層・2層が利用できない場合、または補完として常に使用:
- Google Trends経由のトレンド把握
- YouTube急上昇・人気動画の調査
- ニュースサイト・ブログのバズ記事
- note/はてブのホットエントリ

**検索パターン**:

    {テーマ} トレンド 2026
    {テーマ} バズ site:note.com OR site:b.hatena.ne.jp
    {テーマ} trending site:youtube.com
    {テーマ} viral tiktok OR instagram 2026
    Google Trends {テーマ}

## 調査フロー

### Step 1: バズスキャン（10分）

利用可能な層を上から順に実行:

1. **1層（Grok）が利用可能なら**: X/Twitterで直近1週間のバズを検索
2. **2層（Xpoz）が利用可能なら**: Instagram/TikTok/Redditの関連トレンドを検索
3. **3層（WebSearch）**: Google Trends、YouTube、note/はてブを検索

各プラットフォームから最低3件のバズコンテンツを収集。

### Step 2: パターン抽出（5分）

収集したバズから共通パターンを抽出:

| 分析軸 | 内容 |
|--------|------|
| フォーマット | リスト？ストーリー？比較？チュートリアル？ |
| フック | 何が人を引きつけているか |
| 感情 | 驚き？共感？怒り？笑い？ |
| タイミング | なぜ「今」バズっているか |
| 再現性 | うちで似たコンテンツを作れるか |

### Step 3: コンテンツ案生成（5分）

バズパターンを基に、指定されたType（A or B）に合わせたコンテンツ案を3〜5件生成:

    ## コンテンツ案

    ### 案1: {タイトル案}
    - Type: A / B
    - プラットフォーム: ブログ / X / YouTube / note
    - 元ネタ: {バズったコンテンツのURL or 概要}
    - 差別化: {元ネタとどう違うか}
    - 想定反応: {誰が、なぜ反応するか}

## `/trend-scout` との棲み分け

| | `/trend-scout` | `/trend-research` |
|---|---------------|-------------------|
| レベル | GTM戦略 | コンテンツ制作 |
| 問い | 「どの市場に参入すべきか」 | 「今何がバズっているか」 |
| 出力 | 市場分析 + GTMプラン | バズ分析 + コンテンツ案 |
| 頻度 | 月1回 | 週1〜随時 |
| 実行者 | writer（analyst検証） | writer / x-manager / video-creator |
