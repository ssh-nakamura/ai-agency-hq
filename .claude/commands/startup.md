# CEO起動ルーティン（通常モード）

以下を順番に実行してください。

## 1. MEMORY.md読み込み（最初に必ず実行）
- `.claude/agent-memory/ceo/MEMORY.md` を読む（判断基準 + 学び）
- `docs/ceo-manual.md` を読む（起動手順 + 意思決定フレームワーク）

## 2. ドキュメント読み込み
- `docs/status.md` を読む（KPI + アクション + 収支）

## 2. 軽量セルフチェック（status.mdの内容から判断）
- [ ] 2週間以上「未着手」のまま放置されているアクションがないか？
- [ ] 株主に報告すべき相談事項（承認待ち含む）が溜まっていないか？
- [ ] 前回セッションで約束した作業が完了しているか？

## 3. 報告（簡潔に）

```
**現在地**: Phase X - Step Y
**前回からの変化**: （あれば）
**セルフチェック**: 問題なし / 問題あり（内容）
**最優先アクション**: A-XXX（内容）
**株主への相談事項**: あり/なし
```

## 注意
- 長々と読み上げない。問題がなければ「問題なし」で良い
- フルモード（週次/戦略判断時）は `docs/ceo-manual.md` に記載:
  plan.md読み込み + validate-docs.py実行 + 改善セルフチェック全項目
