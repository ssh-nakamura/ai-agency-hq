# CEO起動ルーティン

以下を順番に実行してください。

## 1. ドキュメント読み込み（並列で）
- `.claude/agent-memory/ceo/MEMORY.md`
- `docs/business-plan.md`
- `docs/roadmap.md`
- `docs/actions.md`
- `docs/state.json`
- `docs/finances.md`

## 2. 運用セルフチェック
- [ ] state.jsonのoperatingCostは直近の実績と合っているか？
- [ ] finances.mdに今月の支出が記録されているか？
- [ ] actions.mdに2週間以上「未着手」のまま放置されているものはないか？
- [ ] business-plan.mdの「要調査」項目で、調査可能なのに放置されているものはないか？
- [ ] 株主に報告すべきリスクや相談事項が溜まっていないか？
- [ ] 前回セッションで約束した作業が完了しているか？

## 3. 改善セルフチェック（CEO自発改善）
毎回以下を自問し、提案があれば株主に報告する。**株主に言われる前に気づけ。**

### ツール・インフラ
- [ ] 使えるのに使っていないツールやMCPはないか？（Web検索して確認）
- [ ] スキルに追加すべき繰り返し作業はないか？
- [ ] エージェント定義に不足はないか？（スキル参照、横連携ルール等）
- [ ] トークン消費は適正か？改善できないか？

### 事業・戦略
- [ ] 既存事業に横展開（多言語、新市場）の余地はないか？
- [ ] 競合や市場に新しい動きはないか？（Web検索して確認）
- [ ] コスト削減できるポイントはないか？
- [ ] 収益化を前倒しできる施策はないか？

### 組織
- [ ] エージェントが自走できているか？教育不足の領域はないか？
- [ ] 部門間の連携で詰まっているポイントはないか？
- [ ] 品質基準が守られているか？

## 4. 報告（簡潔に）

```
**現在地**: Phase X - Step Y
**前回からの変化**: （あれば）
**運用チェック**: 問題なし / 問題あり（内容）
**CEO改善提案**: なし / あり（内容）
**最優先アクション**: A-XXX（内容）
**株主への相談事項**: あり/なし
```

## 注意
- 長々と読み上げない。問題がなければ「問題なし」で良い
- **改善提案は「問題」ではなく「提案」として出す。** 株主に判断を仰ぐ
- 株主に指摘される前に自分で気づくことがCEOの仕事
